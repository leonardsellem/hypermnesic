# Code of Conduct

## Our standard

This project adopts the [**Contributor Covenant**, version 2.1][covenant] as its code
of conduct. The full, canonical text is available at the link below, and we expect all
participants — contributors, maintainers, and community members — to follow it.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: we are committed to a welcoming, respectful, and harassment-free experience
for everyone, regardless of background or identity. We ask everyone to engage with
empathy and good faith, accept constructive feedback gracefully, and focus on what is
best for the community and the project.

## Scope

This Code of Conduct applies within all project spaces — the repository, issues, pull
requests, and discussions — and when an individual is representing the project in
public spaces.

## Reporting

If you experience or witness unacceptable behavior, or have any other concerns, please
report it privately to the project maintainer:

- **Leonard Sellem** — leonard@sellem.me

All reports will be reviewed and investigated promptly and fairly. The maintainer is
obligated to respect the privacy and security of the reporter of any incident. For the
enforcement guidelines we follow, see the [Contributor Covenant 2.1 enforcement
section][covenant].
