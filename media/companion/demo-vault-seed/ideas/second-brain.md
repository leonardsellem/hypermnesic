# Second brain

A trusted external store for the thinking you don't want to re-do. The test is retrieval:
can you find the claim months later, in context?

Built from [[ideas/evergreen-notes]] and kept alive with [[ideas/spaced-repetition]]. The
files matter more than the app — plain markdown you own.
