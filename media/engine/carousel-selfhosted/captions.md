# Carousel — r/selfhosted

**One claim:** your memory is plain markdown on your own disk, version-controlled in
git — no cloud, no vendor database.

| Still | Caption |
|---|---|
| `01-local-files.png` | `ls -R` — your memory is just files in folders you own. |
| `02-plain-markdown.png` | Every memory is a markdown file you can open, diff, and grep. |
| `03-real-git.png` | Its history is a real `git log` — the index is a disposable projection you can delete and rebuild. |

Stills captured with VHS `Screenshot` against the disposable fixture vault.
`ls -R` (not `ls -la`) is used deliberately — the `-l` owner column would print the
recorder's username. OCR-swept clean (no username, owner, home path, or host). See `../../.review-log.md`.
